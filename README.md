# ROSS no OpenSD 2026

O presente repositório contém todos os materiais utilizados na apresentação realizada no **OpenSD 2026** (Open-Source Scientific Computing in Structural Dynamics), intitulada *ROSS: Uma Alternativa Open-Source para
Simulação da Dinâmica de Rotores*.

A apresentação, no formato PDF, pode ser acessada por meio do link: [Apresentação ROSS no OpenSD 2026](https://file.notion.com/f/f/401073b4-20da-4bc5-b6a4-12b2b13fe1da/bfcadd81-a4c6-4b48-bfba-7912783d41f8/apresentacao.pdf?table=block&id=399c488d-01c6-80f3-930b-fc5e5caef182&spaceId=401073b4-20da-4bc5-b6a4-12b2b13fe1da&expirationTimestamp=1783677600000&signature=3QqT-RAg_Zx7QryxC6wbHZtbnMtRbZ89J9HFujUxZfA&downloadName=apresentacao.pdf)

Antes de executar os scripts presentes nesse repositório, certifique-se de ter a biblioteca ROSS instalada em seu ambiente, utilizando o comando abaixo:

```bash
pip install ross-rotordynamics
```

*Nota: O arquivo `rotor_model_lpco2.toml` é um arquivo de modelo auto-gerado e gerenciado automaticamente pela biblioteca ROSS. Não edite este arquivo manualmente.*

## Recursos Adicionais
Para auxiliá-lo no desenvolvimento e compreensão do código, recomendamos a utilização das seguintes ferramentas:

- **[Documentação do ROSS](https://ross.readthedocs.io):** Principal referência para aqueles que pretendem entender mais sobre o ROSS;
- **[ROSS GPT](https://chatgpt.com/g/g-6838c48fbfa081918b61d77b997fdc33-ross-gpt):** Chatbot personalizado que possibilita a exploração da documentação e auxilia o usuário no desenvolvimento de scripts e na análise de resultados.
